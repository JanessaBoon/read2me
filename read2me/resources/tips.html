Read2Me! is as simple as paste text and hit the play button! Enjoy!

Icon Shortcut Action
CTRL+V Paste copied text.
SPACE BAR Read the text / Pause the reading.
SPACE BAR Continue reading when paused.
LEFT ARROW Go back one sentence.
RIGHT ARROW Jump to next sentence.
UP ARROW Go back one paragraph.
DOWN ARROW Jump to next paragraph.
RETURN KEY Stop reading and go back to the top of the text.
Edit mode is ON. You can edit the text.
Edit mode is OFF. You need to press STOP to enter edit mode.
Adjust the volume of the voice with the scrollbar.
Takes effect once you hit the STOP button
Adjust the voice speed with the scrollbar.
Takes effect once you hit the STOP button
Export the text to an audio file.
Handy if you want to listen to something on the go!

For advanced help or more info, see the menu Help -> F.A.Q
